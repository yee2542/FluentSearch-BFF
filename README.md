# FluentSearch Backend for Frontend

## Description

## Installation

install dependencies

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```

## Docker

```bash
$ docker-compose -f docker-compose.dev.yml up --build
```
